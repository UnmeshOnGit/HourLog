# HourLog

HourLog is a minimal and lightweight Android productivity app designed to help users log and track their daily activities hour by hour. The app provides a structured timeline for each day, allowing users to quickly record what they did during specific hours and revisit past logs anytime.

## Overview

The main goal of HourLog is simplicity. Instead of complex productivity systems, tasks, or overwhelming dashboards, the app focuses on one thing:

* Logging your day in a clean hourly format
* Saving entries locally on the device
* Accessing previous daily records easily
* Building consistency and self-awareness through daily tracking

The app is designed for users who want a personal activity journal, routine tracker, productivity log, or simple day history system.

## Features

* Hourly day logging system
* Clean and minimal Android UI
* Separate log sheet for every day
* Quick writing experience for each hour slot
* Local storage support for offline access
* View and access previous day records
* Lightweight and fast performance
* Mobile-first experience

## Use Cases

HourLog can be used for:

* Personal productivity tracking
* Daily journaling
* Time management improvement
* Habit and routine monitoring
